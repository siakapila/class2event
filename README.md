# Class2Event – Club Event Management Platform

Class2Event is a full-stack web application designed to help college clubs organize and manage events efficiently. The platform allows clubs to create events, manage participating teams, track event schedules, and monitor upcoming and past events through a clean dashboard interface.

The system includes authentication, event creation and management, team tracking, and a responsive dashboard to visualize event information.

Built using modern web technologies including React, Node.js, Prisma ORM, and PostgreSQL.
